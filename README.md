## Deploy to GitHub Pages

### Step 1: Create Repository
1. Go to [GitHub](https://github.com/new)
2. Repository name: `chamet-download` (or any name)
3. Make it **Public**
4. Don't initialize with README

### Step 2: Upload Files
**Option A - Web Interface (Easiest)**
1. Click "uploading an existing file"
2. Drag these 3 files:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Click "Commit changes"

**Option B - Git Command Line**
```bash
git init
git add .
git commit -m "Initial commit: Chamet download page"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/chamet-download.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to repository **Settings**
2. Scroll to **"Pages"** (left sidebar)
3. Under **"Source"**:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **Save**
5. Wait 2 minutes
6. Your site is live at: `https://YOUR-USERNAME.github.io/chamet-download/`
